## GitBook本地运行

*   install GitBook

    ```sh
    npm install gitbook-cli -g
    ```

*   Clone

    ```sh
    git clone git@github.com:xiaoyureed/design-patern-note.git
    cd design-patern-note
    gitbook install
    gitbook serve
    ```

*   查看

    浏览器查看: http://localhost:4000
